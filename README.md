# 🚕 Uber Ride Analysis Dashboard (Power BI)

## 📊 Project Overview
This project presents an **interactive Uber ride analysis dashboard** built using **Power BI**. The report explores ride data to uncover patterns, trends, and insights related to booking behavior, trip performance, and operational metrics.

The goal of this project is to transform raw ride data into **meaningful visual insights** that help understand Uber ride trends and support data-driven decision-making.

---

## 🎯 Objectives
- Analyze Uber ride patterns and trends
- Identify peak booking times and high-demand locations
- Understand trip distribution across different categories
- Provide interactive visual insights for easier exploration of ride data

---

## 🛠️ Tools & Technologies
- **Power BI** – Data visualization and dashboard development  
- **Power Query** – Data cleaning and transformation  
- **DAX (Data Analysis Expressions)** – Calculated measures and metrics  
- **Dataset** – Uber ride data

---

## 📌 Dashboard Features
The Power BI dashboard includes:

- 📅 **Time-based Analysis** – Ride trends by day, week, and month  
- 📍 **Location Insights** – Popular pickup and drop-off locations  
- 🚗 **Ride Distribution** – Trip patterns and ride frequency  
- 📈 **Performance Metrics** – Total trips, booking trends, and comparisons  
- 🔍 **Interactive Filters** – Dynamic filtering for deeper analysis
